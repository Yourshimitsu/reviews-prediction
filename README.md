# reviews-prediction
По отзывам амазон предсказываем оценку пользователя 1-5
Pet-project по NLP: предсказание оценки 1–5 по тексту отзывов Amazon (Video Games, 5-core)

1 способ TF-IDF + LogReg

2 способ Word2Vec + charCNN -> BiLSTM, обученная модифицированной кросс-энтропией штраф сильнее за далёкие промахи по шкале + балансировка по частотам классов

3 способ Fine-Tuning BERT
